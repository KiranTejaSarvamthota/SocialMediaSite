FAQ Section

Question 1: Want to Register?

This is a simple process you just have to register with your email address and a password.

Question 2: How to Login?

This is one of the social media site. Where users can create an account and post in thier groups.Other users can see the post they can respond back as a comment or like or dislike buttons.

Question 3: How to Post Messages?

The point of using this website is to create a platform where users/students can clarify their programming related stuff by simply logging in and posting their query. They can also able to add screen shots and other stuff

Question 4: How to add reactions to the Posts?

The users can add reactions to the post either Liking the post or disliking the post. This will add more intercativity to the group posts and the useful posts are given more likes.

Question 5: How to add comments to the Posts?

The users can add comments to the post and continue discussion to the relevant topic, this way user do not have to go through all the unnecessary messages.

Question 6: How to create New groups?

Users are allowed to create new groups.Either the groups could be private or public. The user is able to send invitation to other group members. Other group members can also search for public group and join the group.

Question 7: How to See other User Profiles?

Users are allowed to see other users profile and general information. Only public groups are visible to other users nut not private groups. Also created reputation matrix depending on the user involvement in the groups.

Question 8: Want to Logout?

Users are able to logout from website after posting any content or adding reactions to the posts. After doing necessary tasks the user is able to logout by clicking the logout link
